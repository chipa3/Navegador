Agregar Cliente

En esta pantalla se debe de ingresar los datos del cliente que se desea agregar. Al llenar todos los cuadros de texto debe de presionar el ícono de tarjeta al lado derecho del cuadro de texto de correo, si todos los datos ingresados se validan correctamente entonces se generará un número de tarjeta asociada al cliente. Luego de tener el Número de tarjeta debe presionar el botón de "Agregar" para agregar el cliente con los datos establecidos. Si presiona el botón de "ATRAS" se regresará a la pantalla de "DATOS DE CLIENTE".




Ayuda

La opción de ayuda se muestra con el ícono de pregunta del lado superior izquierdo y tiene la finalidad de dar a conocer el funcionamiento correcto del sistema. Esto con el fin de proporcionar una herramienta que solucione las dudas del usuario relacionadas al sistema.
